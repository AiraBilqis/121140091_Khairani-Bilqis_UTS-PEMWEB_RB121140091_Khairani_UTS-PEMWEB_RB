    Website ini merupakan salah satu tugas sebagai penilaian untuk UTS mata kuliah Pemrograman Web. Pada tugas atau soal UTS yang diberikan mahasiswa harus membuat website yang terdiri dari 3 halaman HTML termasuk dengan penerapan aturan desain layout dan dengan menggunakan aturan CSS. Halaman yang dibuat terdiri dari Halaman Utama (Homepage), Halaman Formulir, dan Halaman Tabel. Setiap halaman file HTML harus memiliki 1 file CSS dengan menggunakan minimal 5 jenis selektor dalam mendesain halaman HTML.

    Pada Homepage ini harus terdapat judul halaman seperti yang tertera pada bagian header diatas termasuk dengan logo ITERA. Kemudian terdapat menu navigasi untuk berpindah ke halaman lain. Pada menu navigasi ini terdapat beberapa menu yaitu menu untuk kembali ke Halaman Utama (Homepage), menu untuk berpindah ke Halaman Formulir, dan menu untuk pindah ke Halaman Tabel. Pada bagian body terdapat sidebar dan konten utama. Untuk konten utama pada halaman ini menjelaskan tentang deskripsi halaman website ini pada "About Task" beserta konten "About Me". Untuk sidebar berisi daftar link yang dapat digunakan untuk mengakses atau melakukan penjelajahan ke halaman eksternal selain dari halaman ini. Pilihan menunya dapat diakses dengan memilih menu di sebelah kanan yang terdiri dari "Menu Utama", "Menu Lainnya", dan "Menu Kontak".

    Ketika saya merasa lelah dan jatuh dalam berkembang, saya selalu melihat video-video motivasi dari para motivator yang selalu memberikan saya semangat kembali. Di bawah ini klip favorit saya.
